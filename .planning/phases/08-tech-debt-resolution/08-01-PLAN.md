     - Reads `s3mo_path_prefix` option internally: `$prefix = get_option('s3mo_path_prefix', 'wp-content/uploads');`

     - Does NOT include MIME type in the returned array (callers add MIME themselves)

  4. Add static method `set_error(int $attachment_id, string $message): void` that:
     - Calls `update_post_meta($attachment_id, self::META_ERROR, $message);`

  5. Add static method `clear_error(int $attachment_id): void` that:
     - Calls `delete_post_meta($attachment_id, self::META_ERROR);`

  6. Update `clear_offload_status()` to also delete META_ERROR:
     - Add `delete_post_meta($attachment_id, self::META_ERROR);` to the method
